# rh-redwebs Website

Your complete multi-page website with 4 theme switcher views!

## What You Have

- **index.html** - Homepage with theme-switching hero sections
- **cookies.html** - Portfolio page (ready for your work)
- **about.html** - About/Services page
- **contact.html** - Contact page with your email & Instagram
- **styles.css** - All 4 themes + page styles
- **theme-switcher.js** - Theme switching logic with localStorage

## The 4 Views

1. **Cutter** (default) - Beige, handwritten, warm creative vibe
2. **Power** - Red/black, aggressive, for sales/business/hustle energy
3. **Vibrant** - Multi-color (pink/purple/gold/blue), sophisticated colorful
4. **Luxury** - Black/gold, refined, expensive/chic aesthetic

## How to Use

1. Open `index.html` in a browser to test
2. Click "Views" button to switch themes
3. Theme persists across pages using localStorage
4. Navigation works across all pages

## To Deploy

**Easy hosting (free):**

1. Go to netlify.com or vercel.com
2. Upload this whole folder
3. Get a live URL instantly

**With a custom domain:**

1. Buy a domain (namecheap.com, ~$12/year)
2. Point it to your Netlify/Vercel site
3. Done!
